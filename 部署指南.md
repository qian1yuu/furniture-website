# 🌐 部署到互联网指南

## 🚀 方法1：Replit（最简单，推荐）

### 步骤1：注册Replit账号
1. 访问 [https://replit.com](https://replit.com)
2. 点击 "Sign Up" 注册账号（免费）

### 步骤2：创建新项目
1. 登录后，点击右上角 "Create Repl"
2. 选择 "Python" 模板
3. 项目名称：`furniture-website`（或你喜欢的名字）
4. 点击 "Create Repl"

### 步骤3：上传文件
在Replit中：
1. 点击左侧 "Files" 面板
2. 删除默认的 `main.py`
3. 点击 "Upload file" 或拖拽文件上传

需要上传的文件：
- `app.py`
- `requirements.txt`
- 整个 `templates/` 文件夹
- 整个 `static/` 文件夹

### 步骤4：修改运行配置
1. 确保 `requirements.txt` 内容：
```text
Flask==3.0.0
Werkzeug==3.0.1
```

2. 在 `app.py` 最后修改（可选）：
```python
if __name__ == '__main__':
    app.run(host='0.0.0.0', port=8080)
```

### 步骤5：运行部署
1. 点击顶部的 "Run" 按钮
2. 等待安装依赖
3. 你的网站地址会显示在页面上！

**网址示例：** `https://furniture-website.yourname.repl.co`

## 🚀 方法2：Render（专业Flask托管）

### 步骤1：准备GitHub仓库
1. 创建GitHub账号
2. 创建新仓库（Public或Private都可以）
3. 上传你的代码到GitHub

### 步骤2：注册Render
1. 访问 [https://render.com](https://render.com)
2. 用GitHub账号注册

### 步骤3：创建Web Service
1. 点击 "New" → "Web Service"
2. 选择你的GitHub仓库
3. 配置：
   - Name: 你的网站名称
   - Runtime: Python 3
   - Build Command: `pip install -r requirements.txt`
   - Start Command: `python app.py`
4. 点击 "Create Web Service"

### 步骤4：修改app.py
```python
if __name__ == '__main__':
    app.run(host='0.0.0.0', port=80)
```

**网址示例：** `https://your-website.onrender.com`
