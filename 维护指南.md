## ⚠️ 常见问题解决

### 网站无法启动
```bash
# 检查 Flask 是否安装
pip list | findstr flask

# 重新安装 Flask
pip install flask
```

### 样式不显示
- 检查 static 文件夹路径
- 清除浏览器缓存（Ctrl+F5）

### 图片不显示
- 检查图片链接是否有效
- 使用本地图片替换网络图片
